# Wuye

互联网是有记忆的。

## 大事记

直面客观事实。

- 2020年10月18日，21时左右开始，物业在未以任何形式通知居民的前提下，直接关闭宿舍供水。

## 意见

提出问题是解决问题的第一步。

- 向全体师生公布各部门职责范围
- 规划自行车电动车停车区域（可以避雨，连廊下不算）
- 规划被褥晾晒区域（一个半月没晒被子了）
- 公开空调使用的安排，为什么我们付了钱却不能自由选择服务
